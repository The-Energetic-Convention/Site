/Users

/Login

Login to a user account. Username and password must be given in multipart/form-data with format: uname (Username) psw (Password)

URL

https://api.thenergeticon.com/Users/Login

HTTP Method

GET

Response Type

Parameters

/Logout

Logout of your account if needed

URL

https://api.thenergeticon.com/Users/Logout

HTTP Method

GET

Response Type

Parameters

/Me

Get your own user info. Use /Users/Login first to make sure the login cookie is set, and sent for this request

URL

https://api.thenergeticon.com/Users/Me

HTTP Method

GET

Response Type

User

Parameters

  • loggedIn String

    The value of the cookie returned from /Users/Login

/User

Get a user by ID

URL

https://api.thenergeticon.com/Users/User/{userid}

HTTP Method

GET

Response Type

SimpleUser

Parameters

  • Path: userid Integer

    The ID of the user to return
